====================
builtin.json
====================

里面写的是系统内置的一些模版

.. code-block:: javascript

    {
      "name": "模版的名称",
      "file": "模版的文件名, 必须跟本文件同一个目录，并且带后缀"
    }


=============================
模版参数
=============================

模版渲染的时候只有两个参数分别为: :code:`get` 和 :code:`data`.

#. :code:`get` 参数来自于 请求的 URL 中的 请求参数 [没有则为 空字典]
#. :code:`data` 参数来自于 请求的 body 中的 JSON [没有则为 None]
